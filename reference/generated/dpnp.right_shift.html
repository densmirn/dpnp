dpnp.right_shift¶

dpnp.right_shift(x1, x2, **kwargs)[source]¶

Shift the bits of an integer to the right.

Bits are shifted to the right x2.

Parameters

  • x1 (array_like or int) – Input arrays or scalars.

  • x2 (array_like or int) – Input arrays or scalars.

  • **kwargs – For other keyword-only arguments.

Returns

out – Output array or scalar if both x1 and x2 are scalars.

Return type

dpnp.ndarray or scalar

Limitations

Parameters x1 and x2 are supported as dpnp.ndarray. Sizes, shapes and data types of input arrays x1 and x2 are supported to be equal. Keyword arguments kwargs are currently unsupported. Otherwise the function will be executed sequentially on CPU. Input arrays x1 and x2 are supported as integer dpnp.ndarray only.

See also

numpy.right_shift

Shift the bits of an integer to the right.

dpnp.left_shift

Shift the bits of an integer to the left.

Examples

>>> import dpnp as np
>>> x1 = np.array([10, 10, 10])
>>> x2 = np.array([1, 2, 3])
>>> out = np.right_shift(x1, x2)
>>> [i for i in out]
[5, 2, 1]
