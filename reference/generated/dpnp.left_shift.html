dpnp.left_shift¶

dpnp.left_shift(x1, x2, **kwargs)[source]¶

Shift the bits of an integer to the left.

Bits are shifted to the left by appending x2 0s at the right of x1.

Parameters

  • x1 (array_like or int) – Input values.

  • x2 (array_like or int) – Input values.

  • x1 – Number of zeros to append to x1. Has to be non-negative.

  • x2 – Number of zeros to append to x1. Has to be non-negative.

Returns

out – Output array or scalar if both x1 and x2 are scalars.

Return type

ndarray or int

See also

right_shift, binary_repr
